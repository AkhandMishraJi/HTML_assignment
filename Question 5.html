What Is Doctype in html ?

This Is not any html tag ,
This is a declaration that we give to browsers that what type of document it is viewing

  • DOctype decaration in html 5 :-
    In html 5 ,doctype declaration is very simple
    <!DOCTYPE html>

  • DOctype decaration in older version of html :-
    In older versions of html ,doctype declaration is very hard
    <DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
