Elements :-

Elements means an opening tag , content and a closing tag, with atributes

Tags :-

Tags Are Of Two Types :-

  1. Container Tags -

    These Tags Contains Opening And Closing Tages and they contains some value . For eg. <p></p>

  2. Empty Tagg Or self Closing Tags -

    These Tags contains only opening tag and atributes . for eg. <img src="image source">
